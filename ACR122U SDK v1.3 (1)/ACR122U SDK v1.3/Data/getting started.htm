System Requirements

This kit will require the following system specifications:

  • IBM compatible Personal Computer with Intel Pentium 800 MHz Processor or higher;
  • Microsoft Windows 2000 / XP / 2003 / Vista;
  • CD-ROM Drive;
  • One available USB port.

Note:

If you have installed previous versions of the ACR122U NFC Reader SDK, then you must uninstall these versions first via Add/Remove Programs in Control Panel before installing this SDK version. Make sure that all previous folders have been deleted.

For non-Vista users, you need to install the .NET Framework to be able to use the ACR122U Tool and Demo Applications. This can be downloaded via: 

Microsoft .NET Framework Version 2.0 (x86) 

http://www.microsoft.com/downloads/details.aspx?FamilyID=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5&displaylang=en

 

Microsoft .NET Framework Version 2.0 (x64) 

http://www.microsoft.com/downloads/details.aspx?familyid=B44A0000-ACF8-4FA1-AFFB-40E78D788B00&displaylang=en

To use the P2P demo, you need to have two ACR122U devices. You may purchase a second ACR122U unit separately.

 

SDK Checklist

The ACR122U NFC Reader SDK contains the following:

  • 1 unit ACR122U NFC Reader
  • 1 unit ACR122T NFC Reader Token
  • 5 Mifare 1K Cards
  • 1 SDK CD-ROM
     

Connection Diagram

To connect the ACR122U device to your PC, plug in the USB connector (Type A) into an available USB port.

Install Smart Card Reader Driver

The ACR122U is a CCID-compliant device. Windows Vista has a built-in Microsoft CCID driver while for Windows 2000 and XP, the driver is downloaded via Windows Update automatically upon smart card reader plug-in. To know more about setting Automatic Updates in your PC, visit http://update.microsoft.com/microsoftupdate/.

You can also install the ACS CCID Driver included in the ACR122U SDK CD-ROM. Click here to view the Driver Installation Guide.

 

Install SDK

Insert the ACR122U SDK CD-ROM into your CD-ROM drive. If the screen below does not appear, run x:\ACR122USDK.EXE where x is the drive letter of your CD-ROM.  

SDK Components

The default path where the SDK will be installed is x:\Program Files\Advanced Card Systems Ltd\ACR122U NFC Reader SDK where x is the drive letter of your local Windows drive. Click here to view the SDK Installation Guide.

Multi-Application Demo

Peer to Peer Demo – The ACR122U NFC Reader can be used in a wide range of applications and this demonstration program simulates a Smart Poster/ NFC Device setting. The application shows conventional functions such as gathering information from a Smart Poster on the go and purchasing Cinema tickets by just tapping onto the poster. This application highlights the NFC capabilities of the ACR122U NFC Reader.

Visitor Management System - This application simulates a Visitor Management System which can be used for trade shows conferences, exhibitions, and other events that require a registration system. The application is divided into two modules, namely Registration and Exhibitor modules.

Sample Codes - Sample codes written in different programming languages are provided, namely Java, Delphi 7, Visual Basic 6, Visual C++ 6, Visual C++ 2005 (x64), Visual C# 2005 and Visual Basic .Net 2005.

The sample codes show the different capabilities of the ACR122U and showcases how to control the ACR122U peripherals and how to communicate with a contactless tag.

Tools and Utilities

ACR122U Tool – The ACR122U Tool is an application utility tool that enables the user to perform reader and card-related commands.

QuickView - The ACS QuickView is a utility program that checks if you have properly installed the ACR122U device.

ACS Easy Key - The ACS Easy Key is a utility program that allows you to change Mifare security settings.

 

ACR122U Scripting Tool – The ACR122U Scripting Tool is designed to work with ACR122U by creating scripts in order to send commands to the contactless cards.
