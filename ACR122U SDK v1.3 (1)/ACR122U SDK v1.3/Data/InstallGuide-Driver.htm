Introduction to PC/SC Driver

The ACS CCID Driver is a Microsoft WHQL-certified driver following the CCID Standard.

Driver Installation Procedure

A step by step guide to installing device drivers:

 

1.  Insert the ACR122U NFC Reader SDK CD-ROM into your CD-ROM drive. If the screen below does not appear, run x:\ACR122USDK.EXE where x is the drive letter of your CD-ROM.

 

 

 

 

 

 
2. Click on "Install ACS CCID Driver".

 

 

 

 

 

 

 

 

 
3. Select the language of your choice and then click “OK”.

 

 

 
4. Please wait while the Windows Installer prepares the installation.

 


 

 
5. Click “Next” to continue with the installation.

 

 

 

 

 

 

 

 

 

 
6. Choose Destination Location.

Setup will install ACS CCID PCSC driver in the following folder x:\Program Files\ACR122U PCSC Driver 1.1.6.2\ where x is the driver letter of your local Windows drive.

To install this folder, click “Next”.

To install to a different folder, click “Browse” and select another folder then click “Next”.

 

 
7. Click “Install”.

 

 

 

 

 

 

 

 

 

 

 
8. Wait for setup to be completed.

 

 

 

 

 

 

 

 

 

 
9. Please plug in the reader then click on “Finish” to exit the program.
